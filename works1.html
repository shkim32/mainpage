개발환경 및 소개

생활코딩 유튜브의 "WEB3 - PHP & MySQL" 동영상을 참고하여 여러 테이블을 이용해 Create(생성), Read(읽기), Update(갱신), Delete(삭제)를 이용해 게시판 기능을 구현하고, PHP 세션을 이용해 로그인 기능까지 구현해 보았습니다.

수행업무

  1. - 게시판 글쓰기, 수정, 삭제 기능 구현
  2. - 게시판 목록 생성
  3. - 로그인, 회원가입 기능 생성

느낀점

MySQL database를 공부하며, 테이블을 이용하여 SQL문법을 사용해보고 PHP를 이용하여 게시판을 구현하며, 테이블과 database 응용을 해볼 수 있었다.
간단한 게시판을 만들어보며 PHP 응용을 한층 더 공부할 수 있었다.

사용기술

  1. - PHP
  2. - MySQL
